이용 방법

1. 테마 소개 페이지 이동

테마 체험을 클릭하면 모든 테마 소개 페이지로 이동합니다.

2. 테마 선택

네비게이션에서 원하는 테마를 선택하면 해당 테마 소개위치로 이동합니다.

3. 테마 실행

원하는 테마를 선택하면 해당 테마 타이틀 옆 바로가기 버튼을 누릅니다.

4. 방 탈출!

선택한 테마의 방에서 탈출하세요!
